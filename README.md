Server Chat Application 💬

Консольное чат-приложение для обмена текстовыми сообщениями по сети между двумя и более пользователями. Реализовано на Java с использованием TCP-сокетов.

## Архитектура

### Потоки

**Сервер:**
- **Главный поток** — бесконечно ждёт новых подключений через `ServerSocket.accept()`
- **ClientHandler (поток на каждого клиента)** — читает сообщения от своего клиента и рассылает их всем остальным через `broadcast()`

**Клиент:**
- **Главный поток** — читает ввод пользователя с консоли и отправляет серверу
- **Поток-получатель** — слушает входящие сообщения от сервера и выводит на экран

### Протокол обмена сообщениями

| Сообщение | Направление | Описание |
|-----------|-------------|----------|
| `JOIN:username` | Клиент → Сервер | Первое сообщение после подключения — передаёт имя |
| `username: текст` | Сервер → Все клиенты | Обычное сообщение от участника |
| `/exit` | Клиент → Сервер | Команда выхода из чата |
| `SERVER: текст` | Сервер → Все клиенты | Служебное сообщение (вход/выход участника) |

## Требования

- Java 17 или выше
- Gradle 8+

## Установка и сборка

Клонируй репозиторий:

```bash
git clone https://git@github.com:BenimaruShiimon/Server-Chat-App.git
cd chatApp
```

Собери JAR-файлы:

```bash
./gradlew serverJar clientJar
```

Файлы появятся в `build/libs/`:
- `server.jar` — сервер
- `client.jar` — клиент

## Настройка

**server-settings.txt** — настройки сервера:
port=12345

**client-settings.txt** — настройки клиента:
host=localhost

port=12345

Измени `host` на IP-адрес сервера если клиент и сервер запущены на разных машинах.

## Запуск

### Через скрипты (рекомендуется)

Запусти сервер:
```bash
./start-server.sh
```

Запусти клиент (в отдельном окне терминала):
```bash
./start-client.sh
```

### Через java -jar напрямую

```bash
java -jar build/libs/server.jar
java -jar build/libs/client.jar
```

### Через Gradle (для разработки)

```bash
./gradlew :com.chat.server.ChatServer.main()
./gradlew :com.chat.client.ChatClient.main()
```

## Использование

1. Запусти сервер
2. Запусти один или несколько клиентов в отдельных окнах терминала
3. Введи своё имя когда появится запрос
4. Пиши сообщения — они появятся у всех участников чата
5. Для выхода напиши `/exit`
   Введите свое имя: Жорик (к примеру)

[2026-06-13 10:11:03] Подключение успешно! Добро пожаловать Жорик в сетевой чат!

--------- ЧАТ НАЧАТ! ЕСЛИ ХОТИТЕ ПОКИНУТЬ ЧАТ ВВЕДИТЕ - /exit ---------

Привет всем!

SERVER: Григорий вошел в чат

Григорий: Привет, Жорик!

/exit

## Логирование

Все события записываются в `file.log` в корне проекта — как на сервере так и на каждом клиенте. Файл дополняется при каждом запуске, данные не теряются.

Пример `file.log`:
[2026-06-13 10:10:38] Сервер запущен на порту 12345

[2026-06-13 10:11:03] Новое подключение: 127.0.0.1

[2026-06-13 10:11:03] Пользователь вошел в чат: Григорий

[2026-06-13 10:11:23] Пользователь вошел в чат: Жорик

[2026-06-13 10:11:34] Григорий: Привет!

[2026-06-13 10:11:47] Жорик: Привет, Григорий!

[2026-06-13 10:12:00] Пользователь вышел из чата: Жорик

## Тесты

Запусти все тесты:

```bash
./gradlew test
```

12 unit-тестов покрывают:
- Чтение порта и хоста из файла настроек
- Поведение при отсутствии файла настроек
- Корректность параметров сервера и клиента
- Запись и дополнение file.log
